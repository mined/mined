Download free source archive

  • mined-2022.26.tar.gz or from the github Releases area.


    Build:
  • Unpack the archive; in its main directory, use make to produce mined.
  • Consult COMPILE.DOC for further options and details.
    • Install:  
  • Run make install for a root installation (for all users).
  • Other options...
  • Run make localinstall to install into /usr/local (or /sw on Mac OS)
  • Run make optinstall to install into /opt/mined
  • Run make install root=$HOME for a personal installation into $HOME/usr/bin and $HOME/usr/share
  • Run make install prefix=$HOME for a personal installation into $HOME/bin and $HOME/share
  • Consult INSTALL.DOC for further options and details.

    • Download binary packages

    Binary packages for the current release will be added to the github release area as available.

    Windows version
    The Windows standalone version is based on the cygwin runtime and is bundled with all dependencies, including the mintty terminal.
    It runs from the Windows explorer context menu for text files, or from the Start menu or a Desktop shortcut.
    It also runs in a Windows console or the Windows Terminal.

    • The Windows self-installing binary package is built with cygwin and includes the cygwin library, the mintty terminal, the dash command shell, and the regtool and cygpath tools (for the installer).
      (For sources for cygwin, mintty, dash see the cygwin repository.)
    MSDOS version
    Runs in DOS emulator (e.g. DOSBox), or on plain DOS (MSDOS/FreeDOS/...).
    Includes enhanced codepage support, also for FreeDOS/DR-DOS.

    • The DOS binary package is built with djgpp.
    VMS version
    Runs on OpenVMS on VAX or ALPHA architecture (or emulator) or Integrity server (Itanium IA64).

    • The VMS binary package is built with DECC and includes an Alpha binary.

    Other installation packages:

    Install Cygwin package

    Run the setup program from the Cygwin home page. In the “Select Packages” menu, open the “Editors” category and select the mined package.

    Install or download openSUSE Linux RPM package

    From the openSUSE software archive M17N project, select a mined package.

    Install or download Linux package

    For CentOS, Debian, Ubuntu, Fedora, RedHat, SUSE (SLE), Univention, select a mined package from the openSUSE software archive, or select a package for your Linux distribution from the openSUSE software archive – mined package page.

    For Slackware, openSUSE, Fedora, ALT Linux, CentOS, select a mined package from pkgs.org.

    Download Solaris package

    For Solaris on SPARC or Intel, packages are available on Sunfreeware.

    Install FreeBSD package

    On a FreeBSD system, type pkg_add -r mined to fetch and install the binary package.
    For the newest version, build and install mined from the source archive.

    Download NetBSD package

    Find a recent NetBSD package on the NetBSD mined page .
    For the newest version, build and install mined from the source archive.

    Install or download Debian package

    On a Debian system, type apt install mined to install the Debian distribution package.
    Or find a more recent Debian package in the openSUSE software archive (see above).

    Download FreeDOS package

    The source and DOS binary archives are also stored here.

    Install Fink package for Mac OS X

    Use the Fink installer.
    Note that on Mac OS X, it is advisable to use either xterm or iTerm 2 as the native Mac terminal application does not support the mouse.
